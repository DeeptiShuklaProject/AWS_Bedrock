# Phase 1 – Core AWS (Beginner)

---

## 📌 Phase Overview
Welcome to **Phase 1 – Core AWS (Beginner)**. This phase serves as a parent navigation container guiding your learning journey through core concepts, production architecture, hands-on labs, and enterprise best practices.

## 🛠️ Skills Covered
VPC Networking, IAM Policies, EC2 Instance Lifecycle, S3 Storage, CloudWatch Observability, RDS Multi-AZ, Route 53 DNS

## 🎓 Learning Objectives
1. **Master** structural architecture and internal mechanisms of services in this phase.
2. **Deploy** production-grade cloud infrastructure following the AWS Well-Architected Framework.
3. **Troubleshoot** real-world production incidents using automated diagnostics, CloudWatch logs, and Boto3 scripts.

## ⏱️ Estimated Duration
4 Weeks

## 📋 Prerequisites
Basic Computer Science and Networking Concepts
