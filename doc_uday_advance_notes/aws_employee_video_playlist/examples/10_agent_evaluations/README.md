# Episode 10 Example: AgentCore Evaluation Frameworks

This directory demonstrates how to write automated test suites to measure AI Agent correctness and latencies prior to launching them into production.

## 🛠️ Concepts Illustrated:
1. **Assertion Assertions**: Setting up mock user queries paired with expected answer keywords (Ground Truth checks).
2. **Performance Benchmarking**: Verifying that agent execution completes within strict service level latencies.
3. **Accuracy Rate Metrics**: Compiling a detailed test summary report to spot failing execution routes.

## 💻 How to Run:
Run the script to print the evaluation summary:
```bash
python 10_correctness_eval.py
```
