# Build Agents on AWS - Step By Step with Demo

Detailed step-by-step walkthrough of the cloud agent building masterclass by Cloud With Raj.

- **Video URL**: [Build Agents on AWS - Step By Step with Demo (YouTube)](https://www.youtube.com/watch?v=epWeri7OQi8)
- **Duration**: 40 Minutes
- **Instructor**: Raj (Cloud With Raj)
- **Key Focus**: Architecting enterprise-ready AI Agents, configuring Action Groups via AWS Lambda, attaching Knowledge Bases, and observing trace outputs in the console.

---

## What is Covered in This Lesson

1. **Enterprise Agent Architecture**: The design principles of generative AI agents on AWS, separating the LLM orchestrator, user session state, and backend tools.
2. **AWS Lambda Integrations**: Writing Lambda function handlers in Python to handle action invocations.
3. **OpenAPI Schema Definitions**: Standardizing tool payloads so the Agent can plan and format requests dynamically.
4. **End-to-End Live Testing**: Triggering the agent, inspecting tracing steps (pre-processing, orchestration, post-processing), and diagnosing security exceptions.
