# Phase 2 – Modern Application Development (Intermediate)

---

## 📌 Phase Overview
Welcome to **Phase 2 – Modern Application Development (Intermediate)**. This phase serves as a parent navigation container guiding your learning journey through core concepts, production architecture, hands-on labs, and enterprise best practices.

## 🛠️ Skills Covered
Serverless Compute, REST & HTTP APIs, NoSQL Modeling, User Pools & OAuth, Asynchronous Decoupling, Pub/Sub, Event Buses

## 🎓 Learning Objectives
1. **Master** structural architecture and internal mechanisms of services in this phase.
2. **Deploy** production-grade cloud infrastructure following the AWS Well-Architected Framework.
3. **Troubleshoot** real-world production incidents using automated diagnostics, CloudWatch logs, and Boto3 scripts.

## ⏱️ Estimated Duration
4 Weeks

## 📋 Prerequisites
Phase 1 Core AWS & Python/JS SDK Basics
