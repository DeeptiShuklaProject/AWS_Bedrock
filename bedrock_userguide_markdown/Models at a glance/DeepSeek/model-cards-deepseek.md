

# DeepSeek
<a name="model-cards-deepseek"></a>

The following DeepSeek models are available in Amazon Bedrock:


| **Model** | **Description** | 
| --- | --- | 
| [DeepSeek V3.2](model-card-deepseek-deepseek-v3-2.md) | DeepSeek V3.2 is DeepSeek's mixture-of-experts model with improved reasoning, coding, and instruction following capabilities. | 
| [DeepSeek-V3.1](model-card-deepseek-deepseek-v3-1.md) | DeepSeek-V3.1 is DeepSeek's 685B parameter mixture-of-experts model with strong performance across coding, math, and general reasoning tasks. | 
| [DeepSeek-R1](model-card-deepseek-deepseek-r1.md) | DeepSeek-R1 is DeepSeek's reasoning model that uses chain-of-thought to solve complex math, coding, and logic problems. |
