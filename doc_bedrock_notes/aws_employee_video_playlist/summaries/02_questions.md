### Question 2: Model Context Protocol (MCP)
**Q**: What is the Model Context Protocol (MCP) and why is it important for agents?
**Answer (Click to expand)**:
<details>
<summary>View Answer</summary>
MCP is an open standard that decouples tool and data source definitions from the agent logic, allowing any LLM client to easily query diverse servers and tools without custom integrations.
</details>
