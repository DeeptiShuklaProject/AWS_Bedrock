

# Pricing and search units
<a name="rerank-pricing"></a>

Reranking is priced per query. A query is a single call to the reranker model that can contain up to 100 document chunks. The `SearchUnits` metric in Amazon CloudWatch represents the number of queries processed by the reranker model.

For current pricing details, see the [Amazon Bedrock pricing page](https://aws.amazon.com/bedrock/pricing/).
