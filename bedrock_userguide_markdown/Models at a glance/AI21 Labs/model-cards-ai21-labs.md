

# AI21 Labs
<a name="model-cards-ai21-labs"></a>

The following AI21 Labs models are available in Amazon Bedrock:


| **Model** | **Description** | 
| --- | --- | 
| [Jamba 1.5 Large](model-card-ai21-labs-jamba-1-5-large.md) | Jamba 1.5 Large is AI21 Labs' hybrid SSM-Transformer model with 398B total parameters and a 256K context window, optimized for complex reasoning tasks across long documents. | 
| [Jamba 1.5 Mini](model-card-ai21-labs-jamba-1-5-mini.md) | Jamba 1.5 Mini is AI21 Labs' lightweight hybrid SSM-Transformer model with 52B total parameters and a 256K context window, optimized for low-latency enterprise tasks. |
