Lesson 01: Terraform Introduction Walkthrough

1. Prerequisites: Install the Terraform CLI and AWS CLI. Configure credentials using `aws configure`.
2. Directory Setup: Create `main.tf` with the S3 bucket configuration.
3. Execution Commands:
   ```bash
   terraform init
   terraform plan -out=tfplan
   terraform apply tfplan
   ```
4. Verification: Check the AWS S3 Console to verify the bucket has encryption enabled and public access blocked.
