

# Stability AI
<a name="model-cards-stability-ai"></a>

The following Stability AI models are available in Amazon Bedrock:


| **Model** | **Description** | 
| --- | --- | 
| [Stable Image Remove Background](model-card-stability-ai-stable-image-remove-background.md) | Stable Image Remove Background is Stability AI's model that accurately removes backgrounds from images, isolating the foreground subject. | 
| [Stable Image Search and Replace](model-card-stability-ai-stable-image-search-and-replace.md) | Stable Image Search and Replace is Stability AI's model that finds objects in images and replaces them with new content based on text prompts. | 
| [Stable Image Style Guide](model-card-stability-ai-stable-image-style-guide.md) | Stable Image Style Guide is Stability AI's model that generates images matching a reference style while following text prompt instructions. | 
| [Stable Image Style Transfer](model-card-stability-ai-stable-image-style-transfer.md) | Stable Image Style Transfer is Stability AI's model that applies the artistic style of a reference image to a target image. | 
| [Stable Image Fast Upscale](model-card-stability-ai-stable-image-fast-upscale.md) | Stable Image Fast Upscale is Stability AI's model that quickly increases image resolution with minimal processing time. | 
| [Stable Image Search and Recolor](model-card-stability-ai-stable-image-search-and-recolor.md) | Stable Image Search and Recolor is Stability AI's model that identifies objects in images and changes their colors based on text prompts. | 
| [Stable Image Conservative Upscale](model-card-stability-ai-stable-image-conservative-upscale.md) | Stable Image Conservative Upscale is Stability AI's image upscaling model that increases resolution while preserving the original image's details and style. | 
| [Stable Image Erase Object](model-card-stability-ai-stable-image-erase-object.md) | Stable Image Erase Object is Stability AI's model that removes unwanted objects from images and fills the area with contextually appropriate content. | 
| [Stable Image Control Structure](model-card-stability-ai-stable-image-control-structure.md) | Stable Image Control Structure is Stability AI's model that generates images guided by structural inputs like depth maps and edge detection. | 
| [Stable Image Creative Upscale](model-card-stability-ai-stable-image-creative-upscale.md) | Stable Image Creative Upscale is Stability AI's model that upscales images while adding creative detail and enhancing visual quality. | 
| [Stable Image Control Sketch](model-card-stability-ai-stable-image-control-sketch.md) | Stable Image Control Sketch is Stability AI's model that generates images guided by sketch inputs for controlled image creation. | 
| [Stable Image Outpaint](model-card-stability-ai-stable-image-outpaint.md) | Stable Image Outpaint is Stability AI's model that extends images beyond their original boundaries with contextually coherent content. | 
| [Stable Image Inpaint](model-card-stability-ai-stable-image-inpaint.md) | Stable Image Inpaint is Stability AI's model that fills in masked regions of images with contextually appropriate content based on text prompts. |
