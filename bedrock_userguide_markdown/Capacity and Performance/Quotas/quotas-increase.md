

# Request an increase for Amazon Bedrock quotas
<a name="quotas-increase"></a>

Quota increase requests in Amazon Bedrock differ depending on which endpoint you use. To request an increase, see the section that applies to your endpoint:
+ For `bedrock-runtime` quotas, see [Requesting a quota increase](quotas-runtime.md#quotas-runtime-increase).
+ For `bedrock-mantle` quotas, see [Requesting a quota increase](quotas-mantle.md#quotas-mantle-increase).
