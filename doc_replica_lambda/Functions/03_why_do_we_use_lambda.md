# Section 3 – Why do we use Lambda?

<a name="sec-3"></a>

### Common Business Use Cases
* **Real-time File Processing**: Generating image thumbnails or validating metadata immediately after a file is uploaded to Amazon S3.
* **Extract, Transform, Load (ETL)**: Parsing transaction logs from S3, transforming format schemas, and importing them into Amazon DynamoDB.
* **Web Application Backends**: Providing API logic for user authentication, product catalogs, and order creation.
* **Scheduled Automation Tasks**: Running cron operations to stop non-production EC2 instances overnight or cleanup temporary S3 resources daily.
* **IoT Data Ingestion**: Processing sensor telemetry data streams in real time from AWS IoT Core.
