### Question 1: Memory Distinction
**Q**: What is the difference between short-term memory and long-term memory in AgentCore?
**Answer (Click to expand)**:
<details>
<summary>View Answer</summary>
* **Short-Term Memory**: Stores immediate conversational state (turns) within a single session.
* **Long-Term Memory**: Summarizes key preferences and contexts across sessions, persisted in databases like DynamoDB.
</details>
