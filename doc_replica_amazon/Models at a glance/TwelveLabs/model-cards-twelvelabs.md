

# TwelveLabs
<a name="model-cards-twelvelabs"></a>

The following TwelveLabs models are available in Amazon Bedrock:


| **Model** | **Description** | 
| --- | --- | 
| [Marengo Embed 3.0](model-card-twelvelabs-marengo-embed-3-0.md) | Marengo Embed 3.0 is TwelveLabs' video embedding model that generates vector representations of video content for search and retrieval. | 
| [Pegasus v1.2](model-card-twelvelabs-pegasus-v1-2.md) | Pegasus v1.2 is TwelveLabs' video-to-text generation model that produces detailed descriptions, summaries, and answers about video content. | 
| [Marengo Embed v2.7](model-card-twelvelabs-marengo-embed-v2-7.md) | Marengo Embed v2.7 is TwelveLabs' video embedding model for multimodal video understanding, search, and classification. |
