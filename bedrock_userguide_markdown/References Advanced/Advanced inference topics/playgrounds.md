

# Generate responses in the console using playgrounds
<a name="playgrounds"></a>

The content from this page has been incorporated into the Amazon Bedrock console experience. To experiment with foundation models, open the [Amazon Bedrock console](https://console.aws.amazon.com/bedrock/) and use the playgrounds directly.

**Note**  
Custom models (such as fine-tuned image generation models) are not supported in the console playgrounds. To use custom models with provisioned throughput, make API calls programmatically. For more information, see [Making inference requests](inference.md).

For information on making inference requests programmatically, see [Making inference requests](inference.md).
